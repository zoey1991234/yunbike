# yunbike

# 概述
 用Java 開發校園租車 Android 應用,資料庫用MySQL來儲存用戶資訊,使用者可以掃描QRcode租借腳踏車

# 環境配置

開發環境：Windows 10　，　Android Studio Jellyfish // 2023.3.1

# 運行步驟

安裝必要項目
點擊Sync Project with Gradle Files
